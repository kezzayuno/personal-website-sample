This is a very simple website for the Personal Website project. 

In order to run it, there are three options: 

1. In your terminal, run "start index.html" (without quotations)
2. Through Node.js 
    1. Install Node.js 
    2. In CMD, run the command npm install http-server -g 
    3. In CMD, navigate to the specific path of your project folder 
    4. In CMD, run the command http-server 
    5. Go to your browser and type localhost:8080
3. Run the command in CMD, python -m http.server 
    - In your browser search up: localhost:8000
